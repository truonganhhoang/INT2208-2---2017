# Nhóm BKA
Ứng dụng học tiếng Nhật miễn phí

Gồm các chức năng chính:
  - chọn các bài học lessons gồm nhiều từ
  - chọn nghĩa tiếng việt của từ
  - xem kết quả đã học được

Công cụ sử dụng:
  - Bootstrap
  - HTML & CSS, Javascript
  - Rails 5
